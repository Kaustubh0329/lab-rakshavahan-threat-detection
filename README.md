# RakshaVahan: Virtual Lab for On-Vehicle Threat Detection and Sensor Fusion

## Overview

RakshaVahan is a comprehensive virtual laboratory for understanding multi-sensor fusion, real-time signal processing, and embedded system design for on-vehicle threat detection. This lab simulates end-to-end electronics, sensing, and advanced sensor-fusion techniques for detecting potential threats to moving vehicles using radar, camera, and LiDAR modalities.

## Lab Objectives

1. **Understand Multi-Sensor Fusion**: Learn how to combine data from multiple sensors (radar, camera, LiDAR) for robust threat detection
2. **Signal Processing Mastery**: Gain hands-on experience with radar signal processing, image analysis, and range measurement
3. **Embedded Systems Design**: Design and implement state machine-based control systems for real-time threat response
4. **Automotive Integration**: Learn CAN bus communication protocols and vehicle interface design
5. **Defense Technology Application**: Understand practical applications of sensor fusion in defense and autonomous vehicles

## Key Features

- **8 Interactive Experiments**: Covering power systems, sensors, signal processing, fusion algorithms, and integration
- **MATLAB/Simulink Simulations**: Industry-standard tools for modeling complex systems
- **Circuit Simulations**: Proteus/Multisim models for hardware verification
- **Real-time Scenarios**: End-to-end virtual mission scenarios demonstrating complete threat detection workflows
- **Comprehensive Documentation**: Theory, procedures, observations, and inference sections for each experiment

## Experiments

### Hardware Subsystems
1. **Power Supply and Protection for On-Vehicle Electronics** - Understand voltage regulation, protection circuits, and automotive power distribution
2. **Radar Sensor Interface and Signal Processing** - Learn radar principles, signal acquisition, and Doppler processing

### Sensor Processing
3. **Camera Sensing and AI-Based Trigger Generation** - Image processing, threat classification, and trigger algorithms
4. **LiDAR Range Measurement and Cross-Check** - 3D range measurement and sensor validation techniques

### Fusion and Control
5. **Multi-Sensor Fusion and Threat Score Computation** - Kalman filtering, data fusion algorithms, and threat scoring
6. **Embedded Controller State Machine for Threat Handling** - Real-time control logic and state management

### Integration and Testing
7. **Vehicle Interface via CAN and Driver Alerts** - CAN protocol implementation and alert system design
8. **End-to-End Virtual Mission Scenario** - Complete system integration and performance testing
